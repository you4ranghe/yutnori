# 윷놀이 온라인 🎲

친구들과 각자 폰·PC에서 초대 코드로 모여 함께 즐기는 실시간 윷놀이입니다.
Supabase(무료) + Vercel(무료)만으로 운영됩니다.

## 구성 파일

| 파일 | 역할 |
|---|---|
| `index.html` | 게임 화면 전체 (로비 → 대기실 → 게임) |
| `config.js` | Supabase 접속 정보 (직접 입력 필요) |
| `schema.sql` | Supabase에 한 번 실행하는 DB 스키마 + 서버 함수 |

## 배포 순서 (약 15분)

### 1. Supabase 프로젝트 만들기

1. https://supabase.com 접속 → 가입/로그인 → **New Project**
2. 프로젝트 이름(예: `yutnori`), DB 비밀번호, 리전(**Northeast Asia (Seoul)** 권장) 선택 → 생성

### 2. DB 스키마 설치

1. 대시보드 왼쪽 메뉴 → **SQL Editor**
2. `schema.sql` 파일 내용 **전체**를 붙여넣고 **Run**
3. "Success" 가 나오면 완료 (여러 번 실행해도 안전합니다)

### 3. 익명 로그인 켜기

1. 대시보드 → **Authentication** → **Sign In / Providers**
2. **Anonymous sign-ins** 를 **켜기(Enable)** → 저장
   - 회원가입 없이 닉네임만으로 참가할 수 있게 해주는 설정입니다.

### 4. 접속 정보 입력

1. 대시보드 → **Project Settings** → **API**
2. `config.js` 를 열어 두 값을 채워 넣기:
   - **Project URL** → `SUPABASE_URL`
   - **anon public** 키 → `SUPABASE_ANON_KEY`
   - anon 키는 공개되어도 되는 키입니다 (권한은 DB의 RLS가 통제).

### 5. Vercel에 배포

방법 A — CLI (가장 간단):
```
cd yutnori-online
npx vercel
```
안내에 따라 로그인 → 기본값으로 배포 → 나오는 URL이 게임 주소입니다.
`npx vercel --prod` 로 정식 주소에 올립니다.

방법 B — GitHub 연동:
1. 이 폴더를 GitHub 저장소로 올림
2. https://vercel.com → **Add New Project** → 저장소 선택
3. Framework Preset: **Other**, 설정 그대로 **Deploy**

배포 전에 로컬에서 먼저 확인하려면:
```
cd yutnori-online
npx serve
```
(`file://` 로 직접 열면 동작하지 않을 수 있으니 로컬 서버로 여세요.)

## 노는 방법

1. 한 명이 접속 → 닉네임 입력 → **방 만들기**
2. 대기실의 **초대 코드**(또는 링크)를 친구에게 전송
3. 친구들은 코드 입력으로 참가, 원하는 팀으로 이동 (팀당 1~6명, 최대 12명)
4. 방장이 옵션(말 개수·빽도·특별칸)을 정하고 **놀이 시작**
5. 자기 차례인 사람만 윷을 던지고 말을 옮길 수 있어요. 모든 참가자 화면에 윷 던지기·말 이동이 실시간으로 재생됩니다.

## 운영 관련 메모

- **데이터는 쌓이지 않습니다.** 끝난 방은 1시간 뒤, 모든 방은 24시간 뒤 자동 삭제됩니다 (새 방이 만들어질 때마다 청소가 실행되어 pg_cron 없이도 동작).
- **윷 결과는 서버(DB 함수)에서 생성**되므로 클라이언트 조작으로 원하는 끗수를 만들 수 없습니다.
- **재접속 지원**: 새로고침하거나 잠깐 연결이 끊겨도 같은 브라우저로 다시 열면 하던 방으로 돌아갑니다.
- **자리 비움 대비**: 차례인 사람이 응답이 없으면 방장에게 "⏭ 차례 넘기기" 버튼이 보입니다.
- **Supabase 무료 프로젝트는 1주일간 요청이 없으면 일시정지**됩니다. 대시보드에서 Restore 버튼으로 재개하면 됩니다.
- Vercel 무료(Hobby) 플랜은 비상업 용도 조건이 있습니다. 지인들과의 게임 용도는 문제없습니다.

## 알려진 한계 (무료·간단 구성의 트레이드오프)

- 게임 도중 참가자가 완전히 나가버리면(재접속 불가) 그 사람 차례에서 멈출 수 있습니다 → 방장이 "차례 넘기기"로 진행하세요.
- 게임이 시작된 방에는 새 사람이 들어올 수 없습니다 (시작 전 대기실에서만 참가 가능).
