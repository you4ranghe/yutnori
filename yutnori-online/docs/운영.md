## 완료 기록

### 1단계 · 높음 — 2026-07-22

**1. 실시간 연결 끊김 처리**

- 방 채널 구독에 `onChannelStatus()` 를 붙여 `CHANNEL_ERROR`·`TIMED_OUT`·`CLOSED` 를 감지합니다.
- 끊기면 화면 위에 **📡 연결이 끊겼어요 — 다시 잇는 중…** 표시가 뜨고, 회복되면 안내가 사라집니다.
- **15초마다 서버와 대조**하는 안전망을 넣었습니다. 실시간 소식을 놓쳤어도 서버 상태가 더 최신이면 그때 따라잡습니다. 실시간이 완전히 죽어도 게임은 이어집니다.
- 다른 앱을 보다 돌아오면(`visibilitychange`) 즉시 맞춥니다.
- 방이 이미 삭제됐다면 로비로 내보냅니다.

**2. 방장이 사라져도 게임이 이어지도록**

- 진행이 **25초 넘게 멈추면** 그 방 참가자 중 **가장 먼저 들어온 사람**(방장 제외)이 진행을 이어받습니다. 컴퓨터 두기와 차례 넘기기 모두 가능합니다.
- 서버도 같은 기준(`_may_run`)으로 허용하므로, 클라이언트만 고친 게 아니라 권한 자체가 열립니다.
- 대행자는 4초 간격으로 시도해 서버에 부담을 주지 않습니다.
- 방장이 **나가기 버튼**을 누른 경우는 기존대로 즉시 승계됩니다. 이번 변경은 **창을 그냥 닫은 경우**를 위한 것입니다.

**3. 상태 위조 방지 (서버 검증)**

`push_state` 와 `force_state` 에서 `_check_state()` 로 아래를 확인합니다.

| 막는 것 | 메시지 |
|---|---|
| 다른 놀이의 상태 끼워넣기 | 다른 놀이의 상태예요 |
| 빽도·특별칸 설정 바꾸기 | 놀이 설정은 도중에 바꿀 수 없어요 |
| 참가자 추가·교체 | 참가자 정보는 바꿀 수 없어요 |
| 말 개수 늘리기 | 말 개수는 바꿀 수 없어요 |
| 완주 기록 되돌리기 | 완주 기록을 되돌릴 수 없어요 |
| 거짓으로 이겼다고 하기 | 아직 이긴 상태가 아니에요 |

**남은 한계**: 이동 규칙 전체를 서버에서 다시 계산하지는 않습니다. 말을 몇 칸 옮겼는지까지 검증하려면 서버에 규칙 엔진을 하나 더 두어야 해서, 가족·지인 규모에는 과합니다. 불특정 다수에게 공개한다면 그때 검토합니다.

**확인한 것**: 문법 검사, DOM 참조 검사, 600판 게임 시뮬레이션, `schema.sql` 구조 검사(구분자·begin/end 균형) 모두 통과.

### 2단계 · 중간 — 2026-07-22

**4. 채팅을 그 방 사람만 읽도록 (+ 7. 관전자 수 표시)**

관전자는 서버에 아무 기록이 없어서 "그 방 사람"인지 알 방법이 없었습니다. 그래서 참가자 표에 **역할(`role`)** 을 두고 관전자도 한 줄 기록하도록 바꿨습니다. 이 한 가지로 두 문제가 함께 풀립니다.

- `players.role` 추가 — `player`(놀이 참가) / `spectator`(관전)
- 채팅 읽기 정책을 **그 방에 기록이 있는 사람**으로 제한. 다른 방 대화는 API 로도 볼 수 없습니다
- 관전으로 들어가면 `watch_room()` 으로 기록되고, **관전 N명**이 방 화면과 놀이 화면에 표시됩니다
- 관전자가 팀을 고르면 그대로 참가자로 승격됩니다
- 정원(12명), 시작 조건, 방장 승계, 대행 자격은 모두 **실제 참가자만** 셉니다
- 관전자가 나가면 자기 기록만 지우고 게임에는 손대지 않습니다

**옛 스키마 호환**: 서버에 `role` 이 아직 없어도 깨지지 않습니다. 값이 없으면 참가자로 보고, `watch_room` 호출이 실패해도 조용히 넘어가 기존처럼 관전만 됩니다.

**5. 놀이 기록 전송량 절감**

기록 60줄을 매번 통째로 주고받던 것을, **새로 생긴 줄만** 실어 보내고 각자 화면에서 이어 쌓도록 바꿨습니다. 서버에는 새로 들어온 사람이 흐름을 파악할 만큼 **최근 6줄**만 남깁니다.

| | 이전 | 지금 |
|---|---|---|
| 상태 1회 전송 | 3.8 KB | **2.0 KB** (46% 절감) |
| 무료 5GB 기준 | 6,922판 | **12,874판** |

각 화면의 놀이 기록은 예전처럼 60줄까지 그대로 보입니다.
